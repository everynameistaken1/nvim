# Neovim Configuration
My Neovim configuration files for working with drupal

## *Prerequisites*
- [Linux, general/terminal knowledge](https://www.linux.org/forums/#linux-tutorials.122)
- [Neovim](https://neovim.io/doc/)

## Installation
Run one of the following commands in your terminal:

***SSH***
```
git clone git@github.com:everynameistaken1/nvim.git ~/.config/nvim
```

***HTTPS***
```
git clone https://github.com/everynameistaken1/nvim.git ~/.config/nvim
```

Start nvim and watch the plugin-manager install everything. Don't forget to restart *Neovim* after *phpactor* has successfully been installed by the plugin-manager.

## *NOTE*
You need to have all the dependencies installed in order for all the plugins to work. All dependencies can be found [here](https://github.com/everynameistaken1/drupal/blob/main/drupal/Dockerfile).

The plugin *which-key* is installed. This allows you to see available mappings by simply pressing the spacebar once and waiting.
